# Humanise Prompt

Copy everything below the line and paste it into any LLM. Fill in the bracketed sections.

This prompt does two things in one pass: strips AI language out, then layers in SEO/AEO structure so the human voice and search visibility compound.

---

I have a piece of content that needs to sound more human and be better structured for search and AI visibility. Please rewrite it following the rules below.

**Target keyword (optional):** [what this piece should rank for]

**Content to rewrite:**

[PASTE YOUR CONTENT HERE]

---

## Step 1 — Strip the AI out

Apply these rules to every sentence:

**Kill these words** (replace with the alternative, don't just delete):
- leverage → use
- utilize → use  
- seamless / seamlessly → automatic, or cut
- robust → strong, or cut
- innovative → cut (always)
- implement → do
- delve → go into
- just → cut
- very / really / quite / pretty → cut
- commence → start
- facilitate → help
- numerous → many
- mission-critical → important
- out of the box → cut
- performant → fast and reliable
- game-changing → specific benefit
- disruptive → cut
- agile → cut
- best practices → proven approaches
- arguably → cut
- actually → cut

**Kill these phrases:**
- "I think / I believe / we believe" → state it directly
- "Today, we're excited to..." → cut
- "Let's dive in" → cut, start with the first real sentence
- "In today's fast-paced / ever-evolving landscape of..." → cut, start with the point
- "It's not just X, it's Y" → rewrite as two sentences
- "In conclusion / To summarize / Overall" → cut, make the last sentence the actual point
- "may potentially / it's important to note that / it's worth mentioning" → cut the meta-commentary

**Fix the structure:**
- Replace em-dashes (—) with commas, semicolons, or full stops
- Convert passive voice to active: "was done by" → "[subject] did"
- Add contractions: "do not" → "don't", "will not" → "won't", "it is" → "it's"
- Swap "we" for "you" when addressing the reader
- Convert title-case headings to sentence case

## Step 2 — Layer in SEO/AEO structure

**TLDR:** If there's no TLDR at the top, add one (2–3 sentences). First sentence = direct answer. No preamble.

**H2s:** Make every heading specific and descriptive. No vague headings.

**FAQ:** If there's no FAQ section, add one at the bottom (4–5 questions, each with a direct first-sentence answer).

**Title:** If provided, evaluate it — does it make a specific promise with the keyword near the front? Offer two improved alternatives if not.

## Output

Return:
1. A change log (max 6 bullets: what you changed and why)
2. The full rewritten content

Don't change: the facts, examples, data points, personal stories, or overall length.
