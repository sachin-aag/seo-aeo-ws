# Keyword Cluster Prompt

Copy everything below the line and paste it into any LLM. Fill in the bracketed sections.

---

I want to build a topic cluster for my SEO content strategy. Here are my details:

**Business type:** [e.g. "freelance UX designer", "B2B SaaS for HR teams", "vegan meal prep company"]

**Target audience:** [be specific — who exactly are you trying to reach?]

**Location:** [city / region / country, or "global / online" if not location-based]

**Keywords I'm already ranking for (if any):** [paste or leave blank]

**Content I've already published (if any):** [titles or topics, or leave blank]

---

Please build a complete topic cluster for me:

1. **Primary keyword** — the one keyword my most important page should rank for. Give me the keyword and explain why you chose it (realistic to rank for, clear intent, specific enough to convert).

2. **Topic cluster table** — 6–8 supporting keywords around the primary topic, each with:
   - The keyword phrase
   - Intent type (Informational / Navigational / Transactional / Commercial)
   - Recommended content type (blog post, service page, comparison page, FAQ guide, etc.)

3. **Intent types explained** — give me a quick example of each intent type using my specific niche, so I know how to think about future keywords.

4. **First 3 pieces to write** — based on the cluster, what should I write first? Prioritise:
   - One bottom-of-funnel piece (comparison, review, "best X for Y")
   - One FAQ/informational piece (earns AI citations)
   - One off-page piece (LinkedIn, Reddit, or Quora — written for that platform)
   
   For each piece, give me a specific draft title that makes a clear promise. "My Thoughts on X" is not a title.
