# Meta Optimizer Prompt

Copy everything below the line and paste it into any LLM. Fill in the bracketed sections.

---

I need to write or improve the title tag and meta description for a web page.

**Page topic:** [what is this page about?]

**Primary keyword:** [the main keyword this page should rank for]

**Current title tag (if exists):** [paste it here, or "none"]

**Current meta description (if exists):** [paste it here, or "none"]

**What makes this page valuable / why should someone click?** [what does the reader get from this page that they won't get elsewhere?]

---

Please produce:

**3 title tag options**, each with:
- The exact title (keyword in the first 3 words, 50–60 characters — count them)
- Character count
- One sentence explaining what this option does well

**2 meta description options**, each with:
- The exact description (150–160 characters — count them, keyword included naturally, ends with a hook)
- Character count

**Rules I want you to follow:**
- Title: primary keyword within the first 3 words, 50–60 chars, specific promise, no keyword stuffing
- Meta description: 150–160 chars, keyword appears naturally, ends with a reason to click (not "Click here!" — a specific value statement)
- No passive voice, no generic filler, no "Welcome to..."

If my current title or meta description has problems, flag them specifically (e.g., "your current title is 78 characters and will be cut off after 'Guide to'").
