# 90-Day Plan Prompt

Copy everything below the line and paste it into any LLM. Fill in the bracketed sections.

---

I want a personalized 90-day SEO and GEO action plan. Please be specific — not a generic framework, but an actual schedule I can follow.

**My business:** [what you do and who you serve]

**My current situation:**
- Pages currently indexed (check with `site:yourdomain.com`): [number, or "unknown"]
- Blog posts or content pieces currently live: [number]
- Domain age: [new / 1–2 years / 3+ years]
- Google Search Console set up: [yes / no]
- Any keyword research done before: [yes / no]
- My website URL: [yourdomain.com]

**Known blockers or constraints:**
- Time available per week: [hours per week]
- Budget for paid tools: [€0 / €50/mo / more]
- Writing ability: [comfortable writing myself / need to use AI assistance / will hire writers]
- Technical access: [can edit my website directly / need to go through a developer / use a CMS like WordPress]

**My primary goal for the 90 days:** [e.g. "start appearing for my main keyword," "get my first blog post ranking," "show up in AI search for my category in Berlin"]

---

Please produce:

1. **A summary of my starting point** — based on what I've told you, what's the most critical thing to fix first?

2. **Month 1 — Fix the foundation** — specific weekly actions for weeks 1–4, each with a clear deliverable

3. **Month 2 — Publish and outreach** — specific weekly actions for weeks 5–8

4. **Month 3 — Optimise and set cadence** — specific weekly actions for weeks 9–12, ending with my ongoing monthly routine

5. **Month 1 blockers** — flag anything that needs to be fixed before anything else will work (e.g. if I'm not indexed, nothing else matters)

Make each action specific, not vague. "Work on content" is not an action. "Write an FAQ-structured piece on [topic], publish by end of week 6, request indexing same day" is an action.
