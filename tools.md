# Tools

One pick per job. Free-first.

No tool replaces the fundamentals, but the right tools make the fundamentals faster and more accurate.

---

## Indexing and technical health

**Google Search Console** — free, essential, non-negotiable
The only tool that tells you exactly what Google knows about your site: which pages are indexed, which queries you're appearing for, which pages have crawl errors. Set it up before anything else.
Start here: [search.google.com/search-console](https://search.google.com/search-console)

**PageSpeed Insights** — free
Check your page load speed on mobile. Under 3 seconds is the target. Gives specific fixes, not just a score.
Start here: [pagespeed.web.dev](https://pagespeed.web.dev)

---

## Keyword research

**Google Search itself** — free, often underrated
Type your keyword and look at: autocomplete suggestions, "People also ask," and "Related searches" at the bottom. These are real queries from real people.

**Ubersuggest** — free tier available
Keyword ideas, search volume estimates, and competitor analysis. The free tier is limited but enough to validate a topic cluster.

**Google Keyword Planner** — free (requires Google Ads account)
The original source. More accurate volume data than third-party tools because it comes from Google directly. Requires signing up for Google Ads — you don't need to spend money, just create an account.

## On-page and content auditing

**Screaming Frog SEO Spider** — free up to 500 URLs
Crawls your site and finds: missing title tags, duplicate meta descriptions, broken links, missing H1s, redirect chains. The free version covers most small sites.
Start here: [screamingfrog.co.uk/seo-spider](https://www.screamingfrog.co.uk/seo-spider/)

**View Source (browser)** — free, always available
Ctrl+U / Cmd+U in any browser. The fastest way to check if your metadata is static HTML (visible here) or JS-rendered (not visible here).

## Structured data

**Google Rich Results Test** — free
Paste any URL or code snippet and see if your JSON-LD is valid and eligible for rich results. Fix errors before fixing warnings.
Start here: [search.google.com/test/rich-results](https://search.google.com/test/rich-results)

**Schema.org** — free reference
The authoritative reference for every schema type. If you're not sure what fields are required vs. optional for a schema, check here first.
Start here: [schema.org](https://schema.org)

## Backlinks

**Ahrefs Backlink Checker** — free (limited)
Check how many referring domains point to any URL. The free version shows your top 100 backlinks.
Start here: [ahrefs.com/backlink-checker](https://ahrefs.com/backlink-checker)

**Ahrefs Link Intersect** — paid
Shows you which sites link to your competitors but not to you. This is the most efficient backlink tool for gap analysis. Worth paying for once you have content worth linking to.

## AI visibility monitoring

**ChatGPT (web search mode)** — free tier available
Test how you appear in AI answers. Use the `geo-check` skill for the exact queries to run. Make sure web search is enabled — the default knowledge-only mode won't reflect current content.

**Perplexity** — free
Perplexity cites its sources explicitly, making it easier to see which pages AI tools are pulling from. Run your category + location queries here.

## Writing and content

**Hemingway Editor** — free (browser version)
Paste your content and it highlights: passive voice, overly complex sentences, adverbs. Aim for Grade 7–9 readability for most audiences.
Start here: [hemingwayapp.com](https://hemingwayapp.com)

**Claude Code + this repo's skills** — what you're already using
The `humanise`, `content-rewriter`, and `content-brief` skills in this repo handle most content optimization tasks. No separate tool needed.

---

## What to pay for (when you're ready)

If you're running SEO at any scale and budget isn't an obstacle:

- **Ahrefs** — the best all-in-one for backlinks, keyword research, and competitor analysis
- **Semrush** — stronger on competitor keyword tracking and content gap analysis
- **Positional** — newer tool designed specifically for content-led SEO teams

None of these are necessary to start. Master Google Search Console and Screaming Frog first. They'll tell you most of what you need for the first 6 months.
