# Monthly SEO Checklist

Run this once a month. Blocks out about 2 hours. SEO is a mortgage — pay it monthly, build equity.

---

## Publish (every month)

- [ ] **Piece 1 published** — topic: _______________
- [ ] **Piece 2 published** — topic: _______________
- [ ] Both pieces pass the pre-publish checklist
- [ ] Both pieces indexed (requested via Search Console)

---

## Restructure (every month)

Pick one existing page that gets impressions but not clicks, or traffic but no conversions.

- [ ] **Page selected:** _______________
- [ ] TLDR added or improved
- [ ] H2s made more specific
- [ ] FAQ section added or updated
- [ ] Published and re-indexed

---

  - Page: _______________ Action: _______________
- [ ] **Queries you're ranking for that you didn't target** — new content opportunity?
  - Query: _______________
- [ ] **Pages newly indexed this month:** _____
- [ ] **Any crawl errors to fix?** Yes / No — if yes: _______________

---

## AI visibility spot-check (every month)

Pick 2–3 of your target queries and test them in ChatGPT or Perplexity (web search mode).

- [ ] Query 1: _______________ — mentioned? Yes / No
- [ ] Query 2: _______________ — mentioned? Yes / No
- [ ] Query 3: _______________ — mentioned? Yes / No

If a competitor you haven't seen before shows up — note which of their pages is getting cited and why (structured content? better FAQ? specific data?).

---

## Technical health (every quarter — flag here monthly if something feels off)

- [ ] `site:yourdomain.com` still returns all key pages
- [ ] No "URL not on Google" errors for key pages in Search Console
- [ ] Page speed still under 3 seconds on mobile (check PageSpeed Insights)
- [ ] No broken internal links (check with Screaming Frog free version or Ahrefs)
